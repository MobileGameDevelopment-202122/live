Basic Game

We nearly have a completed game --- a very basic one I will admit, but we will fix that soon --- all we have to do now is update worldController and worldRenderer so that they control and render the board correctly.

Class: WorldController

Fields

Create a reference to store the game board . . .

1
    public Board board;

Method: init

  • Create a new instance of board
  • set the firstPlayer to be an instance of __HumanPlayer using the board.X symbol
  • set the secondPlayer to be an instance of __FirstSpacePlayer using the board.O symbol
  • start the game by calling start method of board.

Method: update

  • If game state of board is PLAYING then call board.move

Method: touchDown

1
2
    @Override
    public boolean touchDown(int screenX, int screenY, int pointer, int button)
  • This method takes in mouse clicks on desktop, or touchDown on Android and maps the position of the click/touch to a cell on the board.
  • Method logic:
1

    If the game state is PLAYING and the current player is human then
        calculate the row coordinate using height and screenY
        calculate the col coordinate using viewportWidth, screenX and width
        if (row,col) represent a valid board position then
            call board.move(row,col)
    return true

Class: WorldRenderer

The only change in this class is in method render. Insert the line

1
    worldController.board.render(batch);

into the render pipeline.
