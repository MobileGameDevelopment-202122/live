(Formative) Assignment

Before we start on multiple screens and general GUI widgets, I want you to investigate basic AI techniques. And implement better computer players than the one I implemented.

Specification

  • Complete the steps 1-5 given in this practical. At this point you should have a basic, but working X and O game.

  • Implement better AI.

  • This AI implementations will be important for the Numerical X and O assignment.

To Submit

In eclipse you create an archive of your project using the following steps:

  • Select File->Export... -> General -> Archive. Click Next.
  • Select all four of the XandO projects.
  • Call the archive XandO.zip.
  • Click on Finish

  • I am assuming the you have not change the other setting from their default values, i,e

    • Save in zip formant (not save in tar format)
    • "Compress contents of file" is selected
    • "Create directory structure for files" is selected.

  • Upload your XandO.zip and a txt (preferably a markdown based, but plain txt is OK) file called README.md summarising the extensions (in this case the AI) that you implemented and the state of each extension.
