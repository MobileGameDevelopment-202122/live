Assets

In this step we will build up the assets we will use to render the board. You are free to replace my assets with any other or even construct the board using primitives.

Build the sprite map

  • Download the source image (I stole from somewhere - source forgotten)

* Using suitable image manipulation software [GIMP](www.gimp.org) create images [x.png](files/x.png), [o.png](files/o.png), and [board.png](files/board.png). * Place generated images in folder __assets_raw/images__ of __XandO-desktop__ project. * Edit __Main__ class of __XandO-desktop__ project so that it builds the sprite map __xando.png__ in project __XandO-android__. * Update the configuration settings to be - title = "X and O" - width = 800 - height = 480 ### Class: __Assets__ * In package __wit.cgd.xando.game__ * implements __Disposable__, and __AssetErrorListener__ ### Fields
